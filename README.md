# .github
UniVPN — это современный Telegram-бот, который помогает быстро купить VPN и получить защищённое подключение без сложных настроек. Доступен промокод START, открывающий 3 дня тестового периода, чтобы оценить скорость, стабильность и удобство сервиса. Подходит для работы, путешествий и ежедневной приватности.
